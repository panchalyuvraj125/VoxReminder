# 🎙️ VoxRemind ProMax

A next-generation, local-first productivity tool and smart reminder application built with React, TypeScript, and Tailwind CSS. VoxRemind is designed for power users who want ultimate control over their time without relying on cloud APIs or subscriptions.

## ✨ ProMax Features

*   **🧠 Local-First Architecture:** 100% of your data lives in your browser (`localStorage`). No external APIs, no cloud sync delays, complete privacy.
*   **⏱️ Deep Work Zen Environment:** Launch a Pomodoro focus session and enter "Zen Mode"—a full-screen, distraction-free interface featuring a mathematically generated ambient **Brown Noise audio engine** (via Web Audio API) to help you hyper-focus.
*   **📅 Day Architect (Visual Timeline):** View your active reminders plotted on a beautiful, interactive daily timeline block. Visually engineer your schedule and easily spot free blocks of time.
*   **🔁 Daily Routines:** Robust support for recurring tasks. Tell it to "Remind me to drink water every 2 hours" or "Har 30 minute stretch karna hai".
*   **🗣️ Bilingual NLP Engine:** Built-in Natural Language Processing that understands both **English** and **Hindi** time inputs (e.g., "in 5 minutes", "2 ghante baad", "at 5:30 PM").
*   **💾 Data Portability:** Full JSON Data Export and Import capabilities. Back up your entire reminder history and settings locally.

## 🚀 Getting Started

### Prerequisites
* Node.js (v18 or higher recommended)
* npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/voxremind.git
   ```
2. Navigate to the project directory:
   ```bash
   cd voxremind
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack
*   **Framework:** React 18
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS + Lucide React (Icons)
*   **Build Tool:** Vite
*   **State Management:** Custom React Hooks (`useReminders`, `usePomodoro`)
*   **Audio:** Native HTML5 Web Audio API
